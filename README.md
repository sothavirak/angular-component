# Sample Angular Component

[![NPM Version][npm-image]][npm-url]
[![NPM Downloads][downloads-image]][downloads-url]

## Install [http-server][]

```sh
$ npm install http-server -g
```

## Run

```sh
$ cd angular-component && http-server .
```

browse http://localhost:8080/
